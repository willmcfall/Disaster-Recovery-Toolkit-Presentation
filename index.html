Disaster Recovery Toolkit

Application Problem Statement

AmRC staff currently do not have a standard approach to management of large-scale disaster recovery programs, leading to inconsistent approaches, inefficient service delivery, and limited accountability for program managers.

Application Use Cases

  • Project Managers able to standarize tools and approaches, improving communication and efficiency
  • Project Managers able to reference application during project management trainings, improving national society capacity enhancement efforts.

Application Tools

  • Frontend:
    Bootstrap, Handlebars
  • Backend:
    Express, Node, Heroku, MySQL, Sequelize, Mocha, Chai
  • General:
    Git, Slack, VSCode, Reveal.JS, ESLint
